# 技能管理功能详细设计

## 1. 功能概述

技能（Skill）是智能体能力的封装单元。一个技能将**指令说明**和**工具集合**组合在一起，定义智能体如何完成某一类任务。应用（App）通过绑定技能来获得相应能力。

### 技能与工具的关系

- **工具（Tool）** 是原子操作：搜索网页、读取文件、调用 API
- **技能（Skill）** 是编排层：定义执行步骤、绑定所需工具、约束输入输出
- **应用（App）** 通过引用技能来组合能力

```
应用 ──引用──▶ 技能 ──绑定──▶ 工具
```

### 技能的核心要素

| 要素 | 说明 |
|------|------|
| 名称 | 技能的显示名称，如"网页搜索"、"数据分析" |
| 分类 | 技能所属类别，用于筛选和管理 |
| 描述 | 一句话描述技能的功能 |
| 技能说明（instruction） | Markdown 格式的详细指令，定义执行步骤、输入输出、注意事项。大模型根据此说明决定如何编排工具调用 |
| 绑定工具 | 技能执行时可使用的工具列表（内置工具、MCP 工具、API 工具） |

## 2. 技能列表页

### 2.1 页面布局

页面顶部为标题区（"技能管理"）和"创建技能"按钮，下方依次为统计摘要行、搜索栏 + 分类筛选栏、技能卡片网格。

### 2.2 统计摘要

页面顶部展示四个统计数据卡片：

| 指标 | 说明 |
|------|------|
| 技能总数 | 系统中所有技能数量 |
| 已启用 | 状态为 `enabled` 的技能数量 |
| 被引用次数 | 所有技能被应用引用的总次数 |
| 分类数 | 当前技能分类的数量 |

### 2.3 搜索与筛选

| 功能 | 说明 |
|------|------|
| 关键词搜索 | 按技能名称或描述进行模糊匹配 |
| 分类筛选 | 按分类过滤：全部 / 文本处理 / 信息检索 / 代码与推理 / 数据处理 / 多模态 / 编排调度 |
| 分类管理 | 点击筛选栏右侧齿轮图标，弹出分类管理弹窗 |

两者可组合使用，同时生效。

### 2.4 技能卡片

技能以卡片网格形式展示（响应式：1-3 列），每张卡片包含：

| 元素 | 说明 |
|------|------|
| 图标 | 闪电图标，渐变背景 |
| 名称 | 技能显示名称 |
| 分类标签 | 带颜色的分类徽章 |
| 状态指示 | 状态圆点 + 文字（已启用 / 已禁用 / 草稿） |
| 描述 | 技能功能的一句话描述 |
| 引用数 | 被多少个应用引用 |
| 工具数 | 绑定了多少个工具 |
| 更新时间 | 最近更新日期 |

点击卡片跳转到技能详情页。悬停时右下角显示"配置"按钮。

### 2.5 分类管理弹窗

弹窗支持对技能分类进行增删改：

| 操作 | 说明 |
|------|------|
| 查看列表 | 展示所有分类，每行显示分类名称和对应技能数 |
| 新增分类 | 底部输入框，输入名称后点击"添加" |
| 编辑分类 | 悬停显示"编辑"按钮，点击后行内编辑，支持回车保存或 ESC 取消 |
| 删除分类 | 悬停显示"删除"按钮，直接删除 |

## 3. 技能详情页

### 3.1 页面结构

顶部操作栏 + 技能信息头卡 + Tab 页签（概览 / 会话记录 / 版本历史）。

### 3.2 顶部操作栏

| 元素 | 说明 |
|------|------|
| 返回按钮 | 返回技能列表页 |
| 编辑技能 | 跳转到编辑页面 |
| 启用 / 禁用 | 根据当前状态切换，已启用时显示"禁用"（红色），反之显示"启用"（绿色） |

### 3.3 技能信息头卡

展示技能的核心信息：

| 字段 | 说明 |
|------|------|
| 名称 | 技能显示名称 |
| 分类标签 | 带颜色的分类徽章 |
| 状态徽章 | 状态圆点 + 文字 |
| 描述 | 功能描述 |
| 创建者 | 创建人姓名 |
| 创建时间 | 创建日期 |
| 最近更新 | 最后更新日期 |
| 引用数 | N 个应用引用 |
| 工具数 | N 个工具 |

### 3.4 Tab：概览

概览页分为三个区域：

#### 绑定工具

左侧面板，展示技能绑定的工具列表。每个工具显示：
- 工具名称
- 来源标签（内置 / MCP / API），带颜色区分

无绑定工具时显示"未配置工具"空状态。

#### 关联应用

右侧面板，展示引用了此技能的应用列表。每个应用显示：
- 应用名称（可点击跳转到应用详情）
- 应用状态（已发布 / 草稿 / 已下线）

无关联应用时显示"暂无应用引用此技能"空状态。

#### 技能说明

全宽区域，以等宽字体展示 Markdown 格式的技能说明（instruction）内容。包括执行步骤、输入输出定义、注意事项等。

### 3.5 Tab：会话记录

展示通过此技能产生的所有会话记录，以表格形式展示：

**表格列：**

| 列 | 说明 |
|----|------|
| 状态 | 成功（绿色）/ 失败（红色） |
| 来源应用 | 触发此技能的应用名称 |
| 用户 | 发起对话的用户或集成系统 |
| 会话主题 | 对话主题摘要 |
| 轮次 | 对话轮次数 |
| 时间 | 会话发起时间 |
| Tokens | 消耗的 Token 数 |
| 时长 | 会话持续时长 |

**功能：**
- **状态筛选**：全部 / 成功 / 失败
- **展开详情**：点击行展开会话对话详情，逐条显示用户消息和 AI 回复，含时间戳
- **分页**：支持翻页

### 3.6 Tab：版本历史

以时间轴形式展示技能的变更历史：

| 字段 | 说明 |
|------|------|
| 版本号 | 如 `v1.3`，最新版本高亮 |
| 日期 | 发布日期 |
| 作者 | 变更作者 |
| 变更说明 | 描述本次变更内容 |

最新版本带"当前版本"标签。

## 4. 创建 / 编辑技能页

### 4.1 页面结构

左侧为主表单区，右侧为工具选择面板。页面底部为操作按钮。

编辑模式下表单自动回填已有数据，且已绑定的工具处于选中状态。

### 4.2 主表单字段

| 字段 | 是否必填 | 说明 |
|------|----------|------|
| 技能名称 | 必填 | 技能的显示名称 |
| 描述 | 选填 | 一句话描述技能功能 |
| 分类 | 必填 | 从已有分类中选择（下拉框） |
| 技能说明 | 必填 | Markdown 格式的详细指令。定义执行步骤、输入输出、注意事项。可手动填写或 AI 自动生成 |

#### AI 生成技能说明

点击技能说明区域上方的"AI 生成"按钮：
- 前置条件：技能名称已填写
- 系统根据技能名称、描述、已选工具自动生成一份结构化的技能说明模板
- 生成内容包括：技能概述、执行步骤、输入/输出定义、注意事项
- 生成后用户可自由编辑修改

### 4.3 工具绑定面板

右侧面板用于选择技能执行时可使用的工具：

**搜索与筛选：**
- 关键词搜索（按名称或描述）
- 来源筛选（全部 / 内置 / MCP / API）

**工具列表：**

每个工具显示：
- 勾选框（已选中高亮）
- 工具名称
- 来源标签（内置 / MCP / API，带颜色）
- 工具描述

**批量操作：**
- "全选当前筛选结果"按钮
- "清空全部已选"按钮

**已选工具区：**

在面板顶部展示已选中的工具数量和工具标签列表，可逐个移除。

### 4.4 测试面板

点击"测试"按钮打开右侧抽屉式测试面板：

**前置条件：**
- 技能名称和技能说明必须已填写

**功能：**
- 模拟对话界面，可输入消息与技能进行测试交互
- 显示 AI 回复内容
- 支持清空会话重新测试
- 面板可随时打开/关闭

### 4.5 操作按钮

| 按钮 | 说明 |
|------|------|
| 取消 | 放弃修改，返回列表页 |
| 保存（编辑模式） | 保存修改 |
| 创建（新建模式） | 创建新技能 |
| 测试 | 打开测试面板 |

## 5. 技能分类

### 5.1 预置分类

| 分类标识 | 显示名称 | 说明 |
|----------|---------|------|
| `text` | 文本处理 | 文本摘要、情感分析、意图识别、多轮对话等 |
| `retrieval` | 信息检索 | 网页搜索、知识库检索等 |
| `code` | 代码与推理 | 代码执行、SQL 生成等 |
| `data` | 数据处理 | 数据分析、文件解析、可视化等 |
| `multimodal` | 多模态 | 图片理解、OCR、语音等 |
| `orchestration` | 编排调度 | 任务编排、流程调度等 |

### 5.2 分类颜色映射

| 分类 | 颜色 |
|------|------|
| 文本处理 | 蓝色（neon） |
| 信息检索 | 青色（cyan） |
| 代码与推理 | 绿色（emerald） |
| 数据处理 | 紫色（violet） |
| 多模态 | 琥珀色（amber） |
| 编排调度 | 玫红色（rose） |

### 5.3 自定义分类

用户可通过分类管理弹窗新增自定义分类。新分类与预置分类享有同等地位，可在筛选和创建技能时使用。

---

## 6. 技能状态

| 状态 | 显示 | 说明 |
|------|------|------|
| `enabled` | 绿色脉冲圆点 + "已启用" | 技能正常可用，应用可引用和调用 |
| `disabled` | 灰色圆点 + "已禁用" | 技能已禁用，应用不可调用 |
| `draft` | 橙色脉冲圆点 + "草稿" | 技能尚未发布，处于编辑中 |

### 示例：网页搜索技能说明

```markdown
## 执行步骤
1. 接收用户查询关键词
2. 调用搜索引擎 API 执行检索
3. 对返回结果进行摘要提取和排序
4. 返回 Top-N 结果及来源链接

## 输入
- 搜索关键词（文本）
- 结果数量（可选，默认 5）

## 输出
- 搜索结果列表（标题、摘要、链接）
- 结果相关度评分

## 注意事项
- 敏感内容需过滤
- 超时设置为 10 秒
```
